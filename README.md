# metal-ui

GUI for metal-stack.

> [!IMPORTANT]
> The interface is currently read-only, which can be changed if someone has the will to do it. It uses the metal-stack V2 API, which is currently WIP.

## Features

- use `~/.metal-stack/config.yaml`
- login to oidc provider
- list metal-stack resources

## Getting Started

- start mini-lab
- `bun install`
- `bun dev`

## Technologies Used

- [Vite](https://vitejs.dev/) - Next generation frontend tooling.
- [Shadcn/ui](https://shadcn-ui.com/) - Beautifully designed components built with Radix UI and Tailwind CSS.
